***Tabut WordPress Theme*** for Portfolio websites.

== Description ==
Tabut is a user-friendly, modern, and responsive theme, built with most popular Bulma CSS with SEO in mind. Tabut is perfect for businesses, professional free-lancers, and creative individuals. Tabut comes with customizable theme options and a very appealing homepage design.
